# shopping_app

A new Flutter project.
shopping application planed for general use and developed using flutter and firebase.
also we used RESTFUL Api.
with a signup/signin screen.
and sending authentication messages to the users.
and users can use it for publish there products and modify it and also buy products of other users.
with order description and products screen can be shown for the user.
also multible providers provided in this application.
